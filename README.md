
# RENTAL CAR

An app for select and booking car for rent. 

## Functionality

- You can look catalog with cars, selected some of them to favorites. 
- You have posibilities to use filter by brand, price, milesage for best search. 
- Also you can view detail page of car and fill BookForm for rent this one.


## Tech Stack

**Client:** React, Next, CSS, TypeScript

**Libraries:** Axios, @tanstack/react-query, react-hot-toast, zustand, modern-normalize


## Deployment

You need to clone repositoriy

```bash
  git clone <SSH code>
```

Go to folder with project 

```bash
  cd rental-car/
```

To install dependencies

```bash
  npm install
```

To run local server do

```bash
  npm run dev
```
## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`NEXT_PUBLIC_API_URL` -- link to backend
